# План презентации: сервис обработки спутниковых снимков

## Аудитория
Комиссия дипломной защиты, преподаватели, технические специалисты и потенциальные пользователи GIS/ДЗЗ-сервисов.

## Цель презентации
Показать, что проект решает практическую задачу автоматизированной обработки больших GeoTIFF/Sentinel-1 данных, имеет понятную архитектуру, масштабируется через Spark и предоставляет результаты через веб-карту.

## Нарратив
От проблемы ручной обработки больших спутниковых снимков переходим к сервисной архитектуре: загрузка данных, хранение в MinIO, события Kafka, параллельная обработка Spark, метаданные PostgreSQL, выдача результатов через FastAPI/Leaflet. Затем показываем данные, алгоритмы конвейера, масштабирование, дашборд, демонстрационный сценарий, результаты и планы развития.

## Визуальная система
Стиль: технологичная карта, тёмный графитовый фон, зелёно-голубые акценты, карточки со скруглениями, диаграммы потоков, нативные графики. Все смысловые элементы должны быть редактируемыми объектами PowerPoint.

## Источники фактуры
README проекта, docker-compose.yml, FastAPI endpoint-ы, Spark processor, producer, схема MinIO/PostgreSQL/Kafka/Spark.

## План проверки
Проверить экспорт PPTX, рендер превью, наличие ключевых текстов как редактируемых объектов и наличие нативного графика для демонстрации линейного роста времени обработки.
